nginx File not found 错误

nginx配置php老是出错，浏览器打开php脚本报File not found错误，最后才发现是$document_root的设置问题。

这个用出现找不到php的错误。因为$document_root的参数是由root html那一行定义的，默认是在/etc/nginx/html/所以把html换成站点根目录就正常了。
