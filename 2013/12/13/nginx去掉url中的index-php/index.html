nginx去掉url中的index.php

使用情境：我想输入www.abc.com/a/1后，实际上是跳转到www.abc.com/index.php/a/1

配置Nginx.conf在你的虚拟主机下添加：

location / {
     if (!-e $request_filename){
          rewrite ^/(.*)$ /index.php/$1 last;
     }
}

如果你的项目入口文件在一个子目录内，则：

location /目录/ {
     if (!-e $request_filename){
          rewrite ^/目录/(.*)$ /目录/index.php/$1 last;
     }
}
