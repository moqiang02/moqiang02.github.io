用boost模块加速你的drupal站

boost模块可以对 html, xml, ajax, css, javascript进行缓存，极大提升游客访问的速度，几乎可以和静态页面媲美。下载boost http://drupal.org/project/boost
使用前你需要设置好drupal的cron.如果你的服务器不支持。可以使用Poormanscron 1.1 or 2.0 http://drupal.org/project/poormanscron

你必须开启了 Clean URLs（简洁链接）Administer （管理）> Site configuration （网站设置）> Clean URLs
将模块上传到sites/all/modules 在Administer 管理> Site building（网站建构） > Modules（模块）启用boost这个模块

管理（Administer） >>网站设置(Site configuration)>> 性能（performance)Boost directories and file extensions可以设置CACHE路径并把这个文件夹设置为可写
将htaccess rules generation里的内容拷贝到.htaccess里
# RewriteBase /
——-拷贝到这里——–
# Rewrite URLs of the form 'x' to the form 'index.php?q=x'.

然后在Boost setting开启BOOST并进行一些设置

如果BOOST不能运行管理（Administer） >>网站设置(Site configuration)>> 性能（performance)>>Boost setting>>Boost Apache .htaccess settings generation修改第一个和第二个设置，尽量用前面的选项，然后保存到htaccess rules generation将新生成的代码替换到原来的，直到BOOST可以运行
修改robots.txt 将Disallow: /boost_stats.php加到Disallow: /CHANGELOG.txt前面，
怎么知道BOOST正常运行呢。登出，以游客身份访问网站。查看HTML代码。出现类似这样的代码就是成功运行了
<!--Page cached by Boost @ 2009-08-11 06:19:58,expires @ 2009-08-11 09:19:58-->
