githubеҚҡе®ў

hexoзі»еҲ—ж•ҷзЁӢпјҡпјҲдёҖпјүhexoд»Ӣз»Қ

д»Җд№ҲжҳҜhexo

hexoжҳҜдёҖдёӘеҹәдәҺNode.jsзҡ„йқҷжҖҒеҚҡе®ўзЁӢеәҸпјҢеҸҜд»Ҙж–№дҫҝзҡ„з”ҹжҲҗйқҷжҖҒзҪ‘йЎөжүҳз®ЎеңЁgithubе’ҢHerokuдёҠгҖӮдҪңиҖ…жҳҜжқҘиҮӘеҸ°ж№ҫзҡ„@tommy351гҖӮеј•з”Ё@tommy351зҡ„иҜқпјҢhexoпјҡ

еҝ«йҖҹгҖҒз®ҖеҚ•дё”еҠҹиғҪејәеӨ§зҡ„ Node.js еҚҡе®ўжЎҶжһ¶гҖӮ
A fast, simple & powerful blog framework, powered by Node.js.

зұ»дјјдәҺjekyllгҖҒOctopressгҖҒWordpressпјҢжҲ‘д»¬еҸҜд»Ҙз”ЁhexoеҲӣе»әиҮӘе·ұзҡ„еҚҡе®ўпјҢжүҳз®ЎеҲ°githubжҲ–HerokuдёҠпјҢз»‘е®ҡиҮӘе·ұзҡ„еҹҹеҗҚпјҢз”ЁmarkdownеҶҷж–Үз« гҖӮжң¬еҚҡе®ўеҚідҪҝз”ЁhexoеҲӣе»әе№¶жүҳз®ЎеңЁgithubдёҠгҖӮ

дёәд»Җд№ҲиҰҒз”Ёhexo

иҝҳжҳҜеј•з”ЁдёӢдҪңиҖ…зҡ„иҜқпјҡ

дёҚеҸҜжҖқи®®зҡ„еҝ«йҖҹ в”Җ еҸӘиҰҒдёҖзңЁзңјйқҷжҖҒж–Үд»¶еҚіз”ҹжҲҗе®ҢжҲҗ
ж”ҜжҢҒ Markdown
д»…йңҖдёҖйҒ“жҢҮд»ӨеҚіеҸҜйғЁзҪІеҲ° GitHub Pages е’Ң Heroku
е·Із§»жӨҚ Octopress жҸ’д»¶
й«ҳжү©еұ•жҖ§гҖҒиҮӘи®ўжҖ§
е…је®№дәҺ Windows, Mac & Linux

жҲ‘еҶҚеҠ еҮ жқЎпјҡ

  • жҳ“з”ЁгҖӮдёҚд»…йғЁзҪІз®ҖеҚ•пјҢе№іж—¶дҪҝз”Ёдёӯд»…йңҖиҰҒhexo new hexo generate hexo server hexo deployеӣӣдёӘе‘Ҫд»ӨгҖӮдёҚеғҸJekyllйңҖиҰҒеҫҲеӨҡз№Ғзҗҗзҡ„gitе‘Ҫд»ӨгҖӮ
  • иҪ»гҖӮж–Үд»¶е°‘гҖҒе°ҸпјҢжҳ“зҗҶи§ЈпјҢж–№дҫҝиҮӘе®ҡд№үгҖӮ
  • з”ЁжҲ·еӨҡгҖӮиҷҪз„¶иө¶дёҚдёҠJekyllе’ҢOctopressпјҢдҪҶйҒҮеҲ°д»Җд№Ҳй—®йўҳйғҪиғҪжҗңзҙўеҲ°зӯ”жЎҲпјҢжҲ–иҖ…жүҫеҲ°еҗҢж ·дҪҝз”Ёhexoзҡ„з”ЁжҲ·иҝӣиЎҢеҸӮиҖғе’Ңе’ЁиҜўгҖӮ

и°ҒиғҪдҪҝз”Ёhexo

иҝҷжҳҜдёҖдёӘе…Қиҙ№ејҖжәҗзҡ„еҚҡе®ўзЁӢеәҸпјҢд»»дҪ•дәәйғҪеҸҜд»ҘдҪҝз”Ёе’Ңдҝ®ж”№гҖӮдҪҶжҳҜдёҚеҗҢдәҺwordpressпјҢhexoз”ұдәҺйңҖиҰҒдҪҝз”ЁGithub,Git,Markdown,Node.jsиҝҷж ·зҡ„е·Ҙе…·пјҢеҘҪеӨҡжҸ’д»¶гҖҒwidgetйғҪйңҖиҰҒиҮӘе·ұе®үиЈ…гҖҒи®ҫзҪ®гҖӮжүҖд»ҘйҖӮеҗҲйӮЈдәӣжңүдёҖе®ҡи®Ўз®—жңәеҹәзЎҖпјҢе–ңж¬ўжҠҳи…ҫзҡ„дәәгҖӮдҪҶжҳҜпјҢдёҚиҰҒжҒҗжғ§пјҢеҸӘиҰҒи·ҹзқҖжң¬ж•ҷзЁӢиө°пјҢе°ұиғҪеҫҲж–№дҫҝең°и®©иҮӘе·ұзҡ„еҚҡе®ўвҖқйЈһиө·жқҘвҖқгҖӮ
