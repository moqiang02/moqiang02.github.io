WEB服务器、网站、域名、IP地址、DNS服务器之间的关系

域名首先指向你的服务器，这个过程叫解析。
服务器分成好多小块，每小块叫一个空间或者一个虚拟主机。
所以当你输入你的域名以后，服务器收到你域名的访问信息，但不知道要打开这么多个小块中的那一个。所以要在你的空间上增加进去你的域名，意思是你这个空间只支持输入你的域名才可以访问，这一过程叫做域名绑定。
只有用域名解析到服务器上，才能实现输入域名可以访问到服务器上的效果，只有把域名绑定到空间上。才能实现输入你的域名直接打开你空间的目的。也就是说解析和绑定二者都是必须的。
解析是在域名管理后台上做的，绑定域名是在DNS服务器上做的，空间管理后台上一般都有这个功能。

花生壳是一个动态域名解析的客户端，一般是私人网站或者小型服务器才会用到的。

大概的原理是这个样子的
我们上网的时候
都会输入一个网址
这个网址是由一串的字母和符号组成的
网址往往和网站的内容有关，很方便记忆
但其实这个网址只是给人看的，计算机可看不懂
那我们是怎么联网的呢？
就是我们输入这个网址之后
我们的计算机吧这个网址发送给域名解析（DNS）服务器
然后DNS服务器将这个网址翻译成相应的IP地址
而这个IP地址，计算机就能看懂了
就可以按照这个IP地址找到对应的网页服务器了

但有个小麻烦
比如我是一个个人用户
我自己开了一个小小的网站
自己开着玩的
网页是挂在我的笔记本上面的
但是我还总需要带着我的笔记本东跑西颠的
在不同的地方上网，IP必然不一样
那怎么才能让别人通过网址很容易找到我现在的IP呢？
就用花生壳这样的软件
我可以注册一个花生壳的帐号
使用花生壳软件
每次登陆的时候
花生壳就会把我当前的IP绑定到我的网址上去
也就是说，无论我的IP怎么变换
其他人都可以通过我的网址找到我当前所在的IP地址。
并登陆我的网站了。
