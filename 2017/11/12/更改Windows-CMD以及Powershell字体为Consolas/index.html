ејҖеҸ‘е·Ҙе…·

жӣҙж”№Windows CMDд»ҘеҸҠPowershellеӯ—дҪ“дёәConsolas

CMDжҳҜCommandзҡ„зј©еҶҷ.еҚіе‘Ҫд»ӨжҸҗзӨәз¬ҰпјҲCMDпјүпјҢжҳҜеңЁOS / 2 пјҢ Windows CEдёҺWindows NTе№іеҸ°дёәеҹәзЎҖзҡ„ж“ҚдҪңзі»з»ҹпјҲеҢ…жӢ¬Windows 2000е’ҢXPдёӯпјҢ VistaдёӯпјҢе’ҢServer 2003 пјүдёӢзҡ„вҖңMS-DOS ж–№ејҸвҖқгҖӮдёӯж–ҮзүҲWindows XP дёӯзҡ„е‘Ҫд»ӨжҸҗзӨәз¬ҰиҝӣдёҖжӯҘжҸҗй«ҳдәҶдёҺDOS дёӢж“ҚдҪңе‘Ҫд»Өзҡ„е…је®№жҖ§пјҢз”ЁжҲ·еҸҜд»ҘеңЁе‘Ҫд»ӨжҸҗзӨәз¬ҰзӣҙжҺҘиҫ“е…Ҙдёӯж–Үи°ғз”Ёж–Үд»¶,еҫ®иҪҜеңЁWindows 7еҸҠд№ӢеҗҺеј•е…ҘдәҶPowershellвҖ”вҖ”дёҖж¬ҫжҜ”CMDжӣҙеҠ ејәеӨ§зҡ„е‘Ҫд»Өе·Ҙе…·гҖӮ

дёӯж–ҮзүҲWindows 10ж“ҚдҪңзі»з»ҹзҡ„CMDд»ҘеҸҠPowershellз•Ңйқўй»ҳи®Өзҡ„еӯ—дҪ“жҳҜж–°е®ӢдҪ“пјҢеҜ№дәҺеҶҷд»Јз Ғзҡ„дәәжқҘиҜҙпјҢзңӢж–°е®ӢдҪ“дёӢзҡ„иӢұж–ҮзӣёеҪ“йҡҫеҸ—пјҢиҝҳжҳҜзңӢеҫ®иҪҜиҮӘеёҰзҡ„ConsolasзңӢиө·жқҘиҲ’жңҚгҖӮ


еңЁWindows 7д№ӢеүҚзҡ„зі»з»ҹдёӯпјҢеҸҜд»ҘзӣҙжҺҘжӣҙж”№CMDзҡ„еӯ—з¬Ұзј–з ҒпјҢдҪҶжҳҜеңЁWindows 10дёӯе·Із»ҸдёҚиғҪзӣҙжҺҘжӣҙж”№еӯ—з¬Ұзј–з ҒдәҶгҖӮ
йӮЈд№ҲпјҢжҲ‘д»¬еҸӘиғҪйҖҡиҝҮжӣҙж”№жіЁеҶҢиЎЁзҡ„ж–№ејҸжқҘдҝ®ж”№CMD е’ҢPowershellеӯ—з¬Ұзј–з ҒгҖӮ

жіЁеҶҢиЎЁдҝ®ж”№еӯ—з¬Ұзј–з Ғ

1.жҢүWindows+Rиҝӣе…ҘRun,иҫ“е…Ҙregedit,еӣһиҪҰиҝӣе…ҘжіЁеҶҢиЎЁгҖӮ

2.еңЁжіЁеҶҢиЎЁең°еқҖж Ҹиҫ“е…Ҙ
HKEY_CURRENT_USER\Console\%SystemRoot%_system32_cmd.exe

3.еӣһиҪҰеҗҺеҸҢеҮ»еҸіиҫ№зҡ„CodePageиҝҷдёҖжіЁеҶҢиЎЁйЎ№пјҢеӢҫйҖүBaseдёӯзҡ„DecimalдёҖйЎ№(зӣ®зҡ„жҳҜе°Ҷж•°еҖјд»ҘеҚҒиҝӣеҲ¶жҳҫзӨә)пјҢз„¶еҗҺе°ҶValue dataдёӯзҡ„ж•°еҖјж”№дёә437пјҢзЎ®и®ӨеҗҺеҶҚжү“ејҖCMDпјҢеҸій”®ж Үйўҳж ҸпјҢйҖүжӢ©еұһжҖ§пјҢйҖүжӢ©еӯ—дҪ“,е°Ҷеӯ—дҪ“ж”№дёәConsolas,дҝқеӯҳеҗҺеӯ—дҪ“е°ұеҸҳжҲҗдәҶConsolasдәҶгҖӮ

еҜ№дәҺPowershell,еңЁжіЁеҶҢиЎЁең°еқҖж Ҹиҫ“е…Ҙ
Computer\HKEY_CURRENT_USER\Console\%SystemRoot%_System32_WindowsPowerShell_v1.0_powershell.exe
еҗҢж ·е°ҶCodePageдёӯзҡ„ж•°еҖјж”№дёә437,дҝқеӯҳеҚіеҸҜгҖӮ

еёёз”ЁеӣҪе®¶е’ҢиҜӯиЁҖжҸҗдҫӣеӯ—з¬ҰйӣҶпјҡ
65001пјҡUTF-8еӯ—з¬ҰйӣҶ
936пјҡз®ҖдҪ“дёӯж–ҮпјҲй»ҳи®Өпјү
437пјҡMS-DOS зҫҺеӣҪиӢұиҜӯ
