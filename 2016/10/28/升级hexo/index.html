升级hexo

因为之前已经升级到3.0.1，所以这次升级过程比较简单。

重新安装了一下hexo-cli：
$ npm update hexo-cli -g

然后，修改了package.json，改为3.1.1：

"dependencies": {
    "hexo": "^3.1.1",

再执行：
$ npm install

然后，用hexo发了一下这篇日志。目前看没有什么问题。
