githubдёӢforkеҗҺеҰӮдҪ•еҗҢжӯҘжәҗзҡ„ж–°жӣҙж–°

githubдёӢпјҢAејҖеҸ‘иҖ…forkдәҶBејҖеҸ‘иҖ…зҡ„йЎ№зӣ®еҗҺпјҢеҰӮжһңBејҖеҸ‘дәәе‘ҳжӣҙж–°д»Јз ҒеҗҺпјҢAејҖеҸ‘иҖ…еҰӮдҪ•иҺ·еҫ—жӣҙж–°пјҹ

еҸӘз”ЁGithubдёҚз”Ёе‘Ҫд»ӨиЎҢзҡ„ж–№жі•

дёҫдҫӢиҜҙжҳҺ: (жҜҸдёҖжӯҘйғҪжңүжҲӘеӣҫ)




е®Ңе…ЁдёҚз”Ёе‘Ҫд»ӨиЎҢзҡ„еҠһжі•пјҢе…¶е®һжҲ‘иҝҳжҳҜжӣҙжҺЁиҚҗе‘Ҫд»ӨиЎҢ

жөҒзЁӢеҰӮдёӢпјҡ

1.йҰ–е…ҲиҰҒе…ҲзЎ®е®ҡдёҖдёӢжҳҜеҗҰе»әз«ӢдәҶдё»repoзҡ„иҝңзЁӢжәҗпјҡ
git remote -v

2.еҰӮжһңйҮҢйқўеҸӘиғҪзңӢеҲ°дҪ иҮӘе·ұзҡ„дёӨдёӘжәҗ(fetch е’Ң push)пјҢйӮЈе°ұйңҖиҰҒж·»еҠ дё»repoзҡ„жәҗпјҡ
git remote add upstream URL
git remote -v
з„¶еҗҺдҪ е°ұиғҪзңӢеҲ°upstreamдәҶгҖӮ

3.еҰӮжһңжғідёҺдё»repoеҗҲе№¶пјҡ

git merge upstream/master
