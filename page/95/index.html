git rm 和 rm 的区别

这是一个比较肤浅的问题，但对于 git 初学者来说，还是有必要提一下的。

用 git rm 来删除文件，同时还会将这个删除操作记录下来；
用 rm 来删除文件，仅仅是删除了物理文件，没有将其从 git 的记录中剔除。
