如何在DIV内只要垂直滚动条，不要水平滚动条

<DIV style="OVERFLOW-Y: scroll; OVERFLOW-X:hidden; width: 685px; height: 180px"><DIV>

说明:

overflow-y: scroll - 有垂直滚动条

overflow-y: auto - 自动

overflow-y: hidden - 无垂直滚动条

宽度width一定不能省略，否则水平方向可以滚动

如果要数字或一个长单词强制换行，可以在style中添加word-break: break-all;属性。

linux php 安装xdebug

我的环境是PHP 5.2.5，下载的xdebug是Xdebug v2.2.1 源码包
PHP 5.3.20用的是Xdebug v2.1.0 {版本一定要匹配}
下载地址为：http://xdebug.org/files/xdebug-2.2.1.tgz
安装过程是：
1、tar zxvf xdebug-2.2.1.tgz
cd xdebug-2.2.1
2、运行 phpize [如果没有将phpize加入$PATH,则应该使用全路径]
phpize
3、然后运行./configure --with-php-config=/usr/local/php/bin/php-config这个根据自己的目录定
之后是 make && make install
