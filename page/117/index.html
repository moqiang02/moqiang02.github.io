и§ЈеҶіEclipseдёӯж–Үд№ұз Ғ

дҪҝз”ЁEclipseзј–иҫ‘ж–Үд»¶з»ҸеёёеҮәзҺ°дёӯж–Үд№ұз ҒжҲ–иҖ…ж–Үд»¶дёӯжңүдёӯж–ҮдёҚиғҪдҝқеӯҳзҡ„й—®йўҳпјҢEclipseжҸҗдҫӣдәҶзҒөжҙ»зҡ„и®ҫзҪ®ж–Үд»¶зј–з Ғж јејҸзҡ„йҖүйЎ№пјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮи®ҫзҪ®зј–з Ғ ж јејҸи§ЈеҶід№ұз Ғй—®йўҳгҖӮеңЁEclipseеҸҜд»Ҙд»ҺеҮ дёӘеұӮйқўи®ҫзҪ®зј–з Ғж јејҸпјҡWorkspaceгҖҒProjectгҖҒContent TypeгҖҒFile
жң¬ж–Үд»ҘEclipse 3.3пјҲиӢұж–ҮпјүдёәдҫӢеҠ д»ҘиҜҙжҳҺпјҡ

  1. и®ҫзҪ®Workspaceзҡ„зј–з Ғж јејҸпјҡ
    Windows->PreferencesвҖҰ жү“ејҖвҖқйҰ–йҖүйЎ№вҖқзӘ—еҸЈпјҢзӮ№еҮ»е·Ұдҫ§еҜјиҲӘж ‘еҲ°General->WorkspaceпјҢеңЁеҸідҫ§и§ҶеӣҫдёӯжүҫеҲ°вҖңText file encodingвҖқйҖүйЎ№и®ҫзҪ®пјҢдёҖз§ҚжҳҜй»ҳи®ӨпјҲDefaultпјүпјҢеҸҰдёҖз§ҚжҳҜд»ҺдёӢжӢүеҲ—иЎЁдёӯйҖүжӢ©пјҲOtherпјүгҖӮEclipse Workspaceй»ҳи®Өзҡ„зј–з Ғж–№ејҸжҳҜж“ҚдҪңзі»з»ҹзҡ„зј–з Ғж јејҸпјҢиҝҷи·ҹж“ҚдҪңзі»з»ҹзҡ„и®ҫзҪ®жңүе…ізі»пјӣеҸҰеӨ–жҲ‘д»¬еҸҜд»ҘйҖүжӢ©OtherеҚ•йҖүжҢүй’®пјҢз„¶еҗҺд»ҺжҢүй’®еҸідҫ§зҡ„дёӢжӢүеҲ—иЎЁдёӯйҖү жӢ©йңҖиҰҒзҡ„зј–з Ғж јејҸпјҲGBKгҖҒISO-8859-1гҖҒUTF-16гҖҒUFT-16 etc.пјүгҖӮ
  2. и®ҫзҪ®Projectзҡ„зј–з Ғж јејҸпјҡ
    еңЁ Workspaceдёӯж–°е»әзҡ„йЎ№зӣ®й»ҳи®Өз»§жүҝWorkspaceзҡ„зј–з Ғи®ҫзҪ®пјҢжҲ‘д»¬д№ҹеҸҜд»ҘеҚ•зӢ¬жӣҙж”№жҹҗдёӘйЎ№зӣ®зҡ„зј–з Ғж јејҸгҖӮеҸій”®зӮ№еҮ»е·ҘзЁӢпјҢйҖүжӢ© PropertiesпјҢжү“ејҖйЎ№зӣ®еұһжҖ§и®ҫзҪ®зӘ—еҸЈпјҢе·Ұдҫ§еҜјиҲӘж ‘йҖүжӢ©ResourceпјҢеңЁеҸідҫ§и§ҶеӣҫдёӯжүҫеҲ°вҖңText file encodingвҖқпјҢдёӨз§Қи®ҫзҪ®йЎ№зӣ®зҡ„зј–з Ғж јејҸпјҢй»ҳи®ӨйҖүдёӯзҡ„жҳҜвҖңInherited from container (XXX)вҖқпјҲжіЁпјҡXXXдёәWorkspaceи®ҫзҪ®зҡ„зј–з ҒпјүпјҢжҲ‘д»¬д№ҹеҸҜд»ҘйҖүжӢ©е…¶д»–зҡ„зј–з Ғж јејҸпјҢи®ҫзҪ®ж–№ејҸеҗҢWorkspaceгҖӮ

Linuxе®үиЈ…PHPеҠ йҖҹеҷЁXcache

XCache жҳҜдёҖдёӘеҸҲеҝ«еҸҲзЁіе®ҡзҡ„ PHP opcoolcode зј“еӯҳеҷЁ. з»ҸиҝҮиүҜеҘҪзҡ„жөӢиҜ•е№¶еңЁеӨ§жөҒйҮҸ/й«ҳиҙҹиҪҪзҡ„з”ҹдә§жңәеҷЁдёҠзЁіе®ҡиҝҗиЎҢ. з»ҸиҝҮ(еңЁlinux дёҠ)жөӢиҜ•е№¶ж”ҜжҢҒжүҖжңүзҺ°иЎҢ PHP еҲҶж”Ҝзҡ„жңҖж–°еҸ‘еёғзүҲжң¬, еҰӮ PHP_4_3 PHP_4_4 PHP_5_0 PHP_5_1 PHP_5_2 HEAD(6.x), е№¶ж”ҜжҢҒзәҝзЁӢе®үе…Ё/Windows. дёҺеҗҢзұ» opcoolcode зј“еӯҳеҷЁзӣёжҜ”жӣҙиғңдёҖзӯ№, жҜ”еҰӮиғҪеӨҹеҝ«йҖҹи·ҹиҝӣ PHP зүҲжң¬.дёӢйқўд»Ӣз»ҚеҰӮдҪ•е®үиЈ…й…ҚзҪ®гҖӮ
1гҖҒе®үиЈ…xcache
wget http://xcache.lighttpd.net/pub/Releases/1.3.2/xcache-1.3.2.tar.gz
tar -zxvf xcache-1.3.2.tar.gz
cd xcache-1.3.2
phpize
./configure вҖ“enable-xcache
make
make install www.2cto.com
