vue-router传递参数3种方法

第一种：get方法

传递值

<router-link :to="{path:'/test',query: { userId: 123,userName:'xia' }}">跳转</router-link>

<router-link :to="{name:'test',query: { userId: 123,userName:'xia' }}">跳转</router-link>
