这里这个返回值是用来判断是否排序成功的
若返回值为负，则表示 A 在排序后的序列中出现在 B 之前。
若返回值为 0，则表示 A 和 B 具有相同的排序顺序。
若返回值为正，则表示 A 在排序后的序列中出现在 B 之后。
